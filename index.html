📋 Setup Instructions

  1. Create a Telegram Bot: Message @BotFather on Telegram and follow the instructions to create a new bot. Copy the bot token.
  2. Get Your Chat ID: Start a chat with your bot, then visit: https://api.telegram.org/bot<YOUR_BOT_TOKEN>/getUpdates to find your chat ID.
  3. Configure: Enter your bot token and chat ID above and click "Save Configuration".
  4. Start Monitoring: Click "Start Camera", allow camera access, then enable "Motion Detection".
  5. Test: Use the "Test Telegram" button to verify notifications are working.
  6. Night Vision: Enable night vision mode for low-light conditions (works best with back camera).
